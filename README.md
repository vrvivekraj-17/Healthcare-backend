# Healthcare Backend Project

This is a Spring Boot backend project for a healthcare management system.

## Features
- Add patient
- View all patients
- View patient by id
- Update patient
- Delete patient

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven

## API Endpoints
- POST /patients
- GET /patients
- GET /patients/{id}
- PUT /patients/{id}
- DELETE /patients/{id}

## Run
1. Open project in Eclipse
2. Configure PostgreSQL in application.properties
3. Run HealthcareApplication.java
4. Test APIs in browser or Postman
